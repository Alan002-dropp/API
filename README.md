# Projeto: API de Notícias

## Sobre o projeto

A gente desenvolveu uma API simples de notícias, onde é possível criar, visualizar, editar e excluir notícias (CRUD).

Também fizemos um app em React Native que se conecta com essa API usando Axios.

---

## Tecnologias usadas

* TypeScript
* Express
* SQLite
* Drizzle ORM
* React Native
* Axios

---

## Como rodar o projeto

### Backend (API)

1. Entre na pasta do backend:

```bash
cd backend
```

2. Instale as dependências:

```bash
npm install
```

3. Rode o projeto:

```bash
npm run dev
```

A API vai rodar em:
http://localhost:3000

---

### Frontend (App)

1. Entre na pasta do app:

```bash
cd mobile
```

2. Instale as dependências:

```bash
npm install
```

3. Rode o app:

```bash
npx expo start
```

---

## O que dá pra fazer

* Criar notícia
* Ver notícias
